# react-native-vision-camera-payment-card-scanner

vision-camera-payment-card-scanner is a production-ready React Native library for fast, native payment card scanning built on top of react-native-vision-camera. It detects payment cards, reads the card number and expiry date, identifies the card brand, validates the PAN with Luhn, and returns structured scan results through a simple TypeScript API.

## Installation


```sh
npm install react-native-vision-camera-payment-card-scanner
```


## Usage


```js
import { multiply } from 'react-native-vision-camera-payment-card-scanner';

// ...

const result = multiply(3, 7);
```
